Scalable Vector Graphics

Hva er SVG?

Scalable Vector Graphics (SVG) er et XML-basert filformat for markeringsspråk som beskriver todimensjonal vektorgrafikk. Det er en åpen standard utviklet og vedlikeholdt av World Wide Web Consortium.

Siden formatet kan benyttes av XML betyr det at det er et tekstbasert bildeformat og i motsetning til punktgrafikkformater som JPEG og GIF er SVG vektorbasert.

For å forstå denne koden trenger man ikke sitte inne på altfor mye kunnskaper når det gjelder farger og bilderedigering. Bildet er bygd opp av separate én- og todimensjonale elementer (linjer og flater) som kan endres og manipuleres hver for seg.

Eksempler på SVG-elementer

Firkant

<html>
<style>
  #firkant { width: 300px;   height: 300px; }

  #formen_til_firkanten {
	     width: 200px;   height: 200px;
	     fill: deeppink;   stroke: white; 
	     stroke-width:5; }
</style>
<body>
  <svg id="firkant">
    <rect x="50" y="20" id="formen_til_firkanten" />
  </svg>
</body>
</html>

Sirkel
